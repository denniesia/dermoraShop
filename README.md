## ✨ Features

- 🛍️ Browse products from API
- 🛒 Add to cart system
- ➕ Quantity management per item
- ❌ Remove items from cart
- 💰 Automatic total price calculation
- ⏳ Loading states for better UX



---

## 🧰 Tech Stack

- ⚛️ React (Vite)
- 🧠 Context API (State Management)
- 🎨 Tailwind CSS
- 🌐 Fetch API
- 🔀 React Router
- ⛓️‍💥 Json Server
