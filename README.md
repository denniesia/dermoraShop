## ✨ Features

- 🛍️ Browse products from API
- 🛒 Add to cart system
- ➕ Quantity management per item
- ❌ Remove items from cart
- 💰 Automatic total price calculation
- ⏳ Loading states for better UX
- 📱 Fully responsive design
